## installation
```shell
git clone https://github.com/afeiship/docify-zip.git
cd docify-zip
npm i && npm link
```

## usage
~~~
__USAGE__
~~~

## user case
1. file with password
   ```shell
   docify-zip -f html_post_test.html -p 123
   ```

2. folder with password
   ```shell
   docify-zip -f snips -p 123
   ```
